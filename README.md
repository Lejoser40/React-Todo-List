## Run Locally

Clone the project

```bash
  git clone https://github.com/Lejoser40/React-Todo-List.git
```

Go to the project directory

```bash
  cd React-Todo-List
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Build project

```bash
  npm build
```
